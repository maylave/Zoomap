# 🦁 Zoomap

**Zoomap** — это современное одностраничное веб-приложение (SPA) для виртуального зоопарка. Проект предоставляет пользователям удобный интерфейс для просмотра галереи животных, покупки билетов онлайн и управления личным профилем.

Приложение разработано с упором на производительность, типизацию и современный дизайн.

## 🚀 Основные возможности

*   **Главная страница**: Приветственный экран с информацией о зоопарке и актуальными новостями.
*   **Галерея животных**: Красивая сетка с фотографиями обитателей зоопарка, описанием видов и интересными фактами.
*   **Покупка билетов**: Интуитивно понятный процесс выбора даты, типа билета и оплаты.
*   **Профиль пользователя**: Личный кабинет с историей покупок, сохраненными билетами и настройками аккаунта.
*   **Адаптивный дизайн**: Полная поддержка мобильных устройств, планшетов и десктопов.

## 🛠 Технологический стек

### Frontend
*   **Framework**: [Vue 3](https://vuejs.org/)
*   **Language**: [TypeScript](https://www.typescriptlang.org/)
*   **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
*   **Build Tool**: Vite
*   **State Management**: Pinia 

### Backend
*   **Runtime**: [Node.js](https://nodejs.org/)
*   **Framework**: Express
*   **Database**: MongoDB / PostgreSQL
